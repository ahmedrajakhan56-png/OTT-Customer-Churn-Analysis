## 🚀 Quick Start Guide

```bash
# 1. Clone the Netflix-style repository
git clone https://github.com/ahmedrajakhan56-png/OTT-Customer-Churn-Analysis.git
cd OTT-Customer-Churn-Analysis

# 2. Install dependencies
pip install -r requirements.txt

# 3. Configure AWS (for cloud analytics)
aws configure
# Enter: AWS Access Key, Secret Key, region (ap-south-1)

# 4. Run ETL pipeline
python src/etl_pipeline.py

# 5. Train ML model
python src/train_model.py

# 6. Launch Power BI dashboard
# Open powerbi/ott_churn_dashboard.pbix
```
